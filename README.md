# AnythingPull

### 万能拖拽viewgroup，Pull-To-Refresh-And-Load

### 下拉刷新 + 上拉加载，支持普通模式弹性模式、层级模式、抽屉模式

### [下载APK把玩](https://raw.githubusercontent.com/TruthKeeper/AnythingPull/master/Sample.apk)

### [简书地址]()

## 如何引入
```
dependencies {
    compile 'com.tk.anythingpull:library:1.0'
}
```
```
allprojects {
    repositories {
        jcenter()
    }
}
```
