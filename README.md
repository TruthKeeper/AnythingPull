# AnythingPull
### 目标：万能拖拽viewgroup
<br>
<br>
#### 将传统的下拉刷新和上拉加载集合弹性滑动
#### 目前支持8种模式
1.下拉刷新
<br>
2.上拉加载
<br>
3.下拉刷新上拉加载
<br>
4.下拉弹性滚动
<br>
5.上拉弹性滚动
<br>
6.下拉弹性滚动，上拉弹性滚动
<br>
7.下拉刷新，上拉弹性滚动
<br>
8.下拉弹性滚动，上拉加载
<br>
